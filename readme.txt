Adding and subtracting integers calculator allows adding and subtracting positive and negative integers and decimals, demonstrating the solution algorithm.

[https://www.calculator.io/integer-calculator/](https://www.calculator.io/integer-calculator/)

== Usage ==

Add the Integer Calculator shortcode to your page, post or sidebar:

`[ci_integer_calculator]`

Add the following code to your template where you would like the Integer Calculator to appear:

`<?php display_ci_integer_calculator(); ?>`

== Installation ==

1. Upload the Integer Calculator /ci_integer_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Integer Calculator plugin through the "Plugins" menu in WordPress.
